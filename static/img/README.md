# 图标设计说明

## 设计理念

本项目的图标设计采用**极简主义**风格,体现"分析思考方法论"的核心概念。

### 视觉元素

#### 1. 同心圆 - 思维层次
- **三层递进的圆环**: 代表思维的深度层次
  - 外层(最浅): 表面观察和初步认知
  - 中层: 深度分析和系统思考
  - 内层(最深): 核心洞察和本质理解

#### 2. 中心点 - 核心思维
- **实心圆点**: 象征思维的核心和焦点
- 代表"第一性原理"和"本质思维"

#### 3. 四个方向连接线 - 多元视角
- **上下左右延伸的线条**: 代表从核心向外扩展的多元视角
  - 上: 宏观视角 / 长期思维
  - 下: 微观细节 / 短期行动
  - 左右: 横向对比 / 多维分析

### 颜色系统

#### 浅色模式 (logo.svg)
- **主色**: #166534 (墨绿) - 自然、理性、优雅
- **辅助色**: #22c55e, #86efac (渐变绿色)
- **背景**: #f0fdf4 (浅绿背景)

#### 暗色模式 (logo-dark.svg)
- **主色**: #4ade80 (亮绿) - 在深色背景下保持可读性
- **辅助色**: #86efac, #bbf7d0
- **背景**: #171717 (深灰)

#### Favicon (favicon.svg)
- 简化版本,适合小尺寸显示
- 保留核心元素:圆环、中心点、四向线条
- 32x32 像素优化

### 文件列表

```

```

### 设计原则

1. **极简主义**: 去除多余装饰,保留核心元素
2. **几何美学**: 使用基础几何图形(圆、线)
3. **寓意清晰**: 每个元素都有明确的象征意义
4. **可扩展性**: 在各种尺寸下都能清晰显示
5. **品牌一致性**: 与网站整体墨绿色调保持一致

### 技术规格

- **格式**: SVG (可缩放矢量图形)
- **浏览器兼容**: 所有现代浏览器
- **尺寸**: 自适应,无失真
- **文件大小**: < 2KB (极小)

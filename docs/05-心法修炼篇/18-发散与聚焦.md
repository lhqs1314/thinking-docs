> 「创造需要发散，决策需要聚焦。」

## 核心理念

**发散与聚焦思维**是指在思考过程中，有意识地在发散（扩展可能性）和聚焦（收敛做选择）两种模式之间切换，而不是始终停留在一种模式。

### 两种思维模式

```
发散思维                          聚焦思维
    │                                 │
    ▼                                 ▼
开放、探索                        收敛、选择
多种可能                          一个决定
创意、联想                        分析、判断
「还有什么？」                    「选哪个？」
扩展选项                          做出选择
```

## 发散思维

### 核心特征

```
□ 延迟判断：先产生想法，不急着评判
□ 追求数量：想法越多越好
□ 鼓励疯狂：奇怪的想法也欢迎
□ 组合嫁接：在已有想法上延伸
```

### 发散技巧

#### 头脑风暴规则

| 规则 | 描述 |
|------|------|
| 不批评 | 任何想法都先接受 |
| 求数量 | 数量优先于质量 |
| 鼓励野想法 | 越疯狂越好 |
| 组合改进 | 在别人想法上延伸 |

#### 联想法

```
核心概念 → 相关联想 → 远距联想 → 跨领域联想

示例（提高团队效率）：
效率 → 流程 → 工厂流水线 → 丰田精益 → 看板系统
效率 → 动力 → 赛车 → 减少阻力 → 减少会议
效率 → 蚂蚁 → 分工协作 → 自组织
```

#### 类比法

```
问题：如何让用户更活跃？

类比其他领域：
- 游戏是如何让玩家上瘾的？→ 成就系统、社交竞争
- 健身房如何留住会员？→ 社群、私教、打卡
- 咖啡店如何让顾客常来？→ 积分、第三空间
```

#### 逆向思维

```
正向：如何成功？
逆向：如何确保失败？然后反过来

示例：
如何让产品失败？
→ 不听用户反馈、功能臃肿、性能差...
→ 反过来：重视反馈、精简功能、优化性能
```

#### SCAMPER法

| 策略 | 问题 |
|------|------|
| S-替代 | 可以用什么替代？ |
| C-组合 | 可以和什么组合？ |
| A-适应 | 可以从哪里借鉴？ |
| M-修改 | 可以如何改变？放大/缩小？ |
| P-其他用途 | 可以用于其他目的吗？ |
| E-消除 | 可以去掉什么？ |
| R-重排 | 可以调换顺序吗？ |

### 发散的障碍与克服

| 障碍 | 克服方法 |
|------|----------|
| 自我审查 | 强制产出数量（如20个想法） |
| 害怕出错 | 创造安全的发散环境 |
| 思维定式 | 强制从不同角度思考 |
| 过早聚焦 | 设定发散时间，到点才能聚焦 |

## 聚焦思维

### 核心特征

```
□ 评估判断：分析各选项优劣
□ 比较权衡：在选项间做取舍
□ 做出选择：最终确定方案
□ 承诺行动：选定后就执行
```

### 聚焦技巧

#### 筛选标准

```
第一轮：可行性筛选
□ 技术上可行吗？
□ 资源上可行吗？
□ 时间上可行吗？

第二轮：价值筛选
□ 影响力大吗？
□ 效果持久吗？
□ 与目标一致吗？

第三轮：优先级排序
→ 选出最高优先级选项
```

#### 决策矩阵

| 选项 | 标准1 | 标准2 | 标准3 | 加权总分 |
|------|-------|-------|-------|----------|
| A | ... | ... | ... | ... |
| B | ... | ... | ... | ... |
| C | ... | ... | ... | ... |

#### 减法原则

```
不是「选最好的」
而是「排除不行的」

逐步排除：
全部选项 → 排除不可行的 → 排除不够好的 → 最终选择
```

#### 直觉检验

```
在理性分析后，用直觉检验：

问自己：
- 这个选择让我兴奋还是担忧？
- 如果硬币决定后我会遗憾吗？
- 这感觉「对」吗？
```

### 聚焦的障碍与克服

| 障碍 | 克服方法 |
|------|----------|
| 决策瘫痪 | 设定决策时限 |
| 完美主义 | 接受「足够好」 |
| 害怕承诺 | 小范围试验后再扩大 |
| 持续发散 | 强制进入聚焦阶段 |

## 模式切换的艺术

### 何时发散

```
□ 问题刚开始，需要理解可能性
□ 现有方案不满意
□ 需要创新突破
□ 时间允许探索
```

### 何时聚焦

```
□ 选项已经足够多
□ 时间压力需要决策
□ 发散变成拖延
□ 需要采取行动
```

### 切换信号

```
发散过度信号：
- 选项太多无法比较
- 开始重复之前的想法
- 陷入分析瘫痪
- 时间消耗过多

聚焦过度信号：
- 只考虑了有限选项
- 方案过于常规
- 忽视了重要可能性
- 做出决定后仍不满意
```

### 保护发散时间

```
错误模式：
发散 → 立刻有人批评 → 发散中止

正确模式：
发散（保护时间）→ 暂停 → 聚焦（评估时间）
```


### 时间盒

```
发散阶段：30分钟
- 只产生想法
- 禁止评判

休息：5分钟

聚焦阶段：30分钟
- 评估筛选
- 做出选择
```

## 常见误区

| 误区 | 描述 | 纠正 |
|------|------|------|
| 永远发散 | 不敢做决定 | 设定聚焦时限 |
| 过早聚焦 | 选项太少就决定 | 强制发散数量 |
| 模式混淆 | 发散时批评 | 分开两个阶段 |
| 假发散 | 只是推迟决定 | 真正探索新可能 |

## 刻意练习

### 日常练习

1. **发散练习**：每天对一个问题强制产生10个想法
2. **聚焦练习**：给自己设定5分钟决策时限
3. **切换意识**：觉察自己当前在哪个模式

### 进阶练习

1. **双钻石实践**：完整走一遍双钻石流程
2. **跨领域类比**：从完全不同领域寻找灵感
3. **逆向思考**：对每个问题尝试逆向思维

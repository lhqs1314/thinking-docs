> 「不要干扰别人的课题，也不要让别人干涉自己的课题。」—— 阿德勒

## 核心理念

**课题分离**是阿德勒心理学的核心概念，指的是明确区分「什么是我的课题」和「什么是别人的课题」，然后专注于自己的课题，不越界干涉他人的课题，也不让他人侵入自己的课题。

### 如何判断是谁的课题

> 「这个选择/行为的后果由谁来承担？」

```
后果的承担者 = 课题的主人

示例：
- 孩子学不学习 → 后果（成绩、未来）孩子承担 → 孩子的课题
- 我是否接受这份工作 → 后果我承担 → 我的课题
- 对方是否喜欢我 → 对方的内心 → 对方的课题
- 我是否努力表达 → 我能控制 → 我的课题
```

## 核心原则

### 一、区分三类课题

```
┌─────────────────────────────────────────────────────────┐
│                        课题类型                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   我的课题              共同课题            他人的课题    │
│   ┌─────────┐         ┌─────────┐         ┌─────────┐  │
│   │我能控制 │         │共同影响 │         │我无法控制│  │
│   │我要承担 │         │需要协商 │         │对方承担 │  │
│   │我负责   │         │共同负责 │         │对方负责 │  │
│   └─────────┘         └─────────┘         └─────────┘  │
│        │                   │                    │       │
│        ▼                   ▼                    ▼       │
│    全力以赴            协商合作             放手/尊重    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 二、专注自己的课题

```
我的课题包括：
□ 我的想法
□ 我的情绪
□ 我的行为
□ 我的选择
□ 我的努力程度
□ 我如何回应

我的责任：
- 为自己的课题负责
- 不逃避自己的课题
- 不把自己的课题推给别人
```

### 三、不干涉他人的课题

```
他人的课题包括：
□ 他人的想法
□ 他人的情绪
□ 他人的评价
□ 他人的选择
□ 他人的人生

边界意识：
- 给建议，但不强求接受
- 表达感受，但不控制对方
- 提供帮助，但不替代负责
- 关心，但不越界
```

### 四、保护自己的课题

```
警惕他人侵入你的课题：
□ 别人的期望在控制你的选择
□ 为了满足他人放弃自己的意愿
□ 承担了本不属于你的责任
□ 被他人的情绪绑架

保护边界：
- 明确表达自己的立场
- 学会说「不」
- 不为他人的情绪负责
- 做自己人生的主人
```

## 实际应用

### 场景一：亲子关系

```
孩子的课题              父母的课题
├── 学习                ├── 提供支持环境
├── 交友                ├── 适度引导
├── 职业选择            ├── 表达关心和建议
└── 人生观              └── 管理自己的焦虑

常见越界：
✗ 强迫孩子按自己意愿选择专业
✗ 替孩子做他该做的决定
✗ 用情绪操控孩子

正确做法：
✓ 表达观点和担忧
✓ 尊重孩子的最终选择
✓ 接受后果由孩子承担
```

### 场景二：职场关系

```
同事的课题              我的课题
├── 他的工作态度        ├── 我的工作表现
├── 他的评价            ├── 我的努力程度
├── 他的情绪            ├── 我的专业成长
└── 他的人际关系        └── 我如何与人协作

常见越界：
✗ 太在意同事对自己的看法
✗ 试图改变同事的态度
✗ 为同事的错误焦虑

正确做法：
✓ 做好自己的工作
✓ 接受无法控制他人评价
✓ 专注可控的部分
```

### 场景三：亲密关系

```
对方的课题              我的课题
├── 对方是否爱我        ├── 我是否真诚表达
├── 对方的感受          ├── 我是否用心经营
├── 对方的选择          ├── 我的期望是否合理
└── 对方的问题          └── 我如何面对结果

常见越界：
✗ 试图控制对方的行为
✗ 为对方的情绪过度负责
✗ 改变自己来讨好对方

正确做法：
✓ 做真实的自己
✓ 表达需求而非要求
✓ 接受对方的独立性
```

## 课题分离的误解

### 不是什么

| 误解 | 澄清 |
|------|------|
| 冷漠 | 不是不关心，是关心但不越界 |
| 自私 | 不是只顾自己，是各负其责 |
| 不沟通 | 不是不表达，是表达但不强求 |
| 放任 | 不是不引导，是引导但不代劳 |
| 回避 | 不是逃避问题，是分清责任 |

### 是什么

```
课题分离 = 清醒的边界意识 + 深切的尊重

既不控制他人，也不被他人控制
既负责自己，也尊重他人的责任
```

## 课题分离的益处

### 对自己

```
□ 减少焦虑（不再为他人课题焦虑）
□ 聚焦精力（专注可控的部分）
□ 提升自主（做自己人生的主人）
□ 降低内耗（不再讨好和纠结）
```

### 对关系

```
□ 减少冲突（不越界就少摩擦）
□ 增进尊重（承认对方的独立性）
□ 健康依赖（相互支持而非绑架）
□ 真实连接（做真实的自己）
```

## 实用工具

### 课题分离检查清单

```
面对一个问题：

1. 这是谁的课题？
   □ 我的课题
   □ 他人的课题
   □ 共同课题

2. 我是否在越界？
   □ 我在干涉他人课题吗？
   □ 我在逃避自己课题吗？

3. 他人是否在越界？
   □ 他人在干涉我的课题吗？
   □ 我需要守护边界吗？

4. 我应该怎么做？
   □ 我的课题：负责到底
   □ 他人课题：放手尊重
   □ 共同课题：协商合作
```

### 课题分离自问

```
□ 这件事的后果由谁承担？
□ 我能控制这件事吗？
□ 我是在帮助还是在控制？
□ 我是在表达还是在强求？
□ 我为什么要为这件事焦虑？
□ 如果这不是我的课题，我该放手什么？
```

## 常见误区

| 误区 | 描述 | 纠正 |
|------|------|------|
| 完全切割 | 以课题分离为由拒绝一切 | 课题分离不是关系隔离 |
| 逃避责任 | 把自己课题推给别人 | 自己的课题必须承担 |
| 过度边界 | 连合理关心都拒绝 | 允许善意进入 |
| 强行分离 | 硬把共同课题分离 | 有些课题需要共同面对 |
| 工具化 | 用来指责他人 | 课题分离是自我修炼 |

## 刻意练习

### 日常练习

1. **课题日记**：每天记录一个「这是谁的课题」的思考
2. **越界觉察**：察觉自己越界或被越界的时刻
3. **放手练习**：刻意放手一件他人课题

### 进阶练习

1. **边界对话**：练习用非攻击性方式守护边界
2. **焦虑分析**：分析焦虑来源是自己还是他人的课题
3. **关系梳理**：梳理重要关系中的课题边界

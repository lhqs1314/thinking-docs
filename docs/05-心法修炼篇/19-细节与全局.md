> 「既见树木，又见森林。」

## 核心理念

**细节与全局思维**是指在分析和执行中，能够在细节关注和全局把握之间灵活切换，既不迷失在细节中忽视大局，也不只看宏观而忽视关键细节。

### 细节与全局的张力

```
细节视角                          全局视角
    │                                 │
    ▼                                 ▼
具体、精确                        抽象、概览
执行层面                          战略层面
「这个bug怎么修」                「系统架构对吗」
风险：一叶障目                    风险：眼高手低
```

## 核心方法

### 一、全局思维

#### 什么时候需要全局

```
□ 开始一个项目/任务时
□ 做重大决策时
□ 发现问题反复出现时
□ 感觉迷失方向时
□ 需要与他人对齐时
```

#### 全局思维的方法

| 方法 | 描述 |
|------|------|
| 后退一步 | 物理或心理上拉开距离 |
| 画大图 | 画出整体结构图 |
| 问目标 | 「我们到底要达成什么？」 |
| 10000米视角 | 假装从高空俯瞰 |
| 终局思维 | 想象完成后的样子 |

#### 全局检查清单

```
□ 我们的核心目标是什么？
□ 现在的方向对吗？
□ 各部分之间的关系是什么？
□ 有什么是我没看到的？
□ 从全局看，优先级对吗？
□ 现在的努力对全局有多大影响？
```

### 二、细节思维

#### 什么时候需要细节

```
□ 进入执行阶段时
□ 质量出现问题时
□ 需要精确无误时
□ 调试排错时
□ 关键节点验收时
```

#### 细节思维的方法

| 方法 | 描述 |
|------|------|
| 放大镜 | 聚焦到最小单元 |
| 检查清单 | 逐项核对 |
| 慢下来 | 减速仔细看 |
| 逐行/逐步 | 一步一步检查 |
| 边界条件 | 检查特殊情况 |

#### 细节检查清单

```
□ 每个步骤都正确吗？
□ 边界情况考虑了吗？
□ 错误处理完善吗？
□ 有遗漏的情况吗？
□ 格式/规范符合要求吗？
□ 能经受挑剔的审视吗？
```

### 三、在两者间切换

#### 切换时机

```
从全局到细节：
- 方向确定后
- 进入执行阶段
- 发现具体问题时

从细节到全局：
- 陷入细节太久时
- 感觉迷失方向时
- 遇到反复出现的问题时
- 需要做取舍决策时
```

#### 切换提醒

```
陷入细节的信号：
□ 在一个小问题上花太长时间
□ 忘记了为什么做这件事
□ 无法判断优先级
□ 感觉「只见树木，不见森林」

忽视细节的信号：
□ 计划很完美但执行出问题
□ 反复出现低级错误
□ 细节被抱怨但不改进
□ 方向对但结果不好
```

### 四、建立连接

#### 从全局到细节的链路

```
全局目标
    │
    ▼
阶段目标
    │
    ▼
任务分解
    │
    ▼
具体行动
    │
    ▼
细节执行

每个层次都能回答「为什么」
```

#### 从细节到全局的验证

```
这个细节 → 服务于什么任务 → 服务于什么目标 → 服务于什么愿景

如果链路断了，可能是：
- 细节不必要
- 或者目标理解有偏差
```

### 五、注意力分配

#### 精力分配原则

```
二八原则：
- 20%精力在全局把控
- 80%精力在细节执行

但也要根据阶段调整：
- 规划阶段：更多全局
- 执行阶段：更多细节
- 复盘阶段：两者兼顾
```

#### 避免两个极端

| 极端 | 表现 | 后果 |
|------|------|------|
| 只看细节 | 完美主义、微观管理 | 效率低、方向偏 |
| 只看全局 | 只谈战略、不落地 | 执行差、结果糟 |

## 实用工具

### 变焦透镜法

```
像相机一样调整焦距：

广角（全局）→ 标准 → 长焦（细节）→ 微距（超细节）

问自己：「现在我应该用什么焦距？」
```

### 分层检视

```
第一层：愿景/目标   □ 清晰吗？对吗？
第二层：策略/路径   □ 合理吗？
第三层：计划/任务   □ 可执行吗？
第四层：具体行动   □ 在做了吗？
第五层：细节质量   □ 做好了吗？

哪一层有问题，就在哪一层聚焦
```

### 定期切换

```
建立切换习惯：

每日：
- 开始工作时：看全局、定优先级
- 工作中：聚焦细节执行
- 结束时：回看全局

每周：
- 周初：全局规划
- 周中：细节执行
- 周末：全局复盘
```

## 常见误区

| 误区 | 描述 | 纠正 |
|------|------|------|
| 细节完美主义 | 每个细节都追求完美 | 区分关键细节和次要细节 |
| 全局空谈者 | 只谈战略不落地 | 战略必须转化为可执行的细节 |
| 切换过频 | 频繁切换导致都做不好 | 在适当时间切换 |
| 忽视层级 | 用细节思维处理全局问题 | 问题在哪个层级解决 |

## 刻意练习

### 日常练习

1. **变焦日记**：每天记录一次视角切换的经历
2. **全局回顾**：工作前花5分钟看全局
3. **细节审视**：对完成的工作做细节检查

### 进阶练习

1. **层级诊断**：问题出现时，判断在哪个层级解决
2. **焦距控制**：刻意练习在不同焦距之间切换
3. **双视角描述**：同一件事用全局和细节两种视角描述
